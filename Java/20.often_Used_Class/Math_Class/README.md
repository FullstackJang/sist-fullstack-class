# Math Class
특징
- 생성자가 없는 클래스
    - 객체화 할 수 없음
- static method로만 구성된 클래스

**절대값**
```java
    Math.abs(수); // -2 => 2
```

**반올림**
```java
    Math.round(실수); // 10.5 => 11
```

**올림**
```java
    Math.ceil(실수);//10.1 => 11
```

**내릴때**
```java
    Math.floor(실수) //10.9 => 10.0
```

**절삭**
```java
// 소수점이하 값을 버릴때는 => Casting 해야함
    (int)10.01; // 10
```

**난수**
```java
    Math.ramdom(); 
```
