과제 내용
```
프로그램이 시작되면 아래의 메뉴를 제공한다. 

 
메뉴 : 1.이름 입력  2.출력   3.종료
입력
 1


1을 입력하고 엔터를 치면  showInputDialog를 제공하여 이름을 
입력받아 java.util.List에 저장한다.
그 후 메뉴를 제공


2를 입력하고 엔터를 치면 showMessageDialog(null, JTextArea)를 
 사용하여 아래와 같이 출력
  이름 
 ------ 
  류수정
  이미주
  서지수


확인을 누르면 닫히면서 메뉴출력.


3을 입력하고 엔터를 치면 showConfirmDialog(null,”정말종료하시겠습니까”);
 confirmDialog에서 “확인” 누르면 Console에 “감사합니다” 를 출력 하고 
프로그램 종료.
```
