# 요소 추가하기

```javascript
    .createElement()
    .createTextNode()
    .appendChild()
```

## 요소 생성

```javascript
    .createElement('tr')
```
위와 같이 작성하면은 아래와 같이 코드가 생성됩니다.
```javascript
    <tr></tr>
```

## 요소에 텍스트 추가

```javascript
    .createTextNode('홍길동');
```

**홍길동**이라는 문자열을 만듭니다.


## 자식요소에 추가

```javascript
        .appendChild()
```
