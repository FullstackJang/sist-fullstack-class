```
"추가"를 누르면 사용자의 입력값을 배열에 추가하고,
"출력"을 누르면 배열의 모든 내용을 순서없는 목록으로 result에 출력하도록 합니다.
```

## 추가
사용자로부터 입력을 받고 추가 버튼클릭 이벤트가 발생하면  
전역변수로 할당 한 `arr배열` 에 `push메소드`를 통해서 사용자가 입력한 값이 저장되게 된다.
